# 🐺 LDL Fitness - Centro de Treinamento

> **Status do Projeto:** 🚀 Em Desenvolvimento / Funcional

O **LDL Fitness** é um ecossistema digital focado em alta performance para alunos de musculação. O projeto une a força bruta do treino raiz com uma interface moderna e intuitiva. O nome carrega os pilares fundamentais da metodologia: **Lealdade, Disciplina e Luta**.

🔗 **Acesse o projeto:** [LDL Fitness Live](https://dudu-airess6.github.io/site-academia/aluno-painel.html)

---

## 🛠️ Tecnologias Utilizadas

Para este projeto, foi utilizada uma estrutura focada em performance e responsividade:

* **HTML5:** Estrutura semântica e acessível.
* **CSS3:** Estilização avançada com Flexbox, CSS Grid e Variáveis globais para manter a identidade visual.
* **JavaScript:** Lógica de navegação e interatividade.
* **Font Awesome:** Biblioteca de ícones para uma interface visual intuitiva.
* **Google Fonts:** Tipografia moderna para leitura clara em dispositivos móveis.

---

## ✨ Funcionalidades

### 🏠 Landing Page Institucional
* Apresentação da trajetória da academia e do Coach Lucas Diniz.
* Seção de serviços detalhando os protocolos exclusivos.
* Design imersivo com fundos de alta qualidade e texturas de academia.

### 🔐 Área do Aluno (Painel Logado)
* **Seleção de Foco:** O aluno pode escolher entre o **Protocolo MASS** (Hipertrofia) ou **Protocolo CUT** (Definição).
* **Painel de Biometria:** Interface dedicada para atualização de dados pessoais e medidas corporais.
* **Central da Matilha:** Página de contato otimizada com links diretos para suporte e redes sociais.
